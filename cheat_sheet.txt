Хеш - это идентификатор коммита
Git хеширует информацию с помощью алгоритма SHA-1 (Secure Hash Algoritm)
40 символьная строка состоящаяя из цифр 0-9 и букв a-f 

git log - полная иформация о коммите
git log --oneline - сокращенная информация


Файл HEAD — один из служебных файлов папки .git. Он указывает на коммит, который сделан последним (то есть на самый новый).
Когда делаеv коммит, Git обновляет refs/heads/master — записывает в него хеш последнего коммита.
Получается, что HEAD тоже обновляется, так как ссылается на refs/heads/master.
